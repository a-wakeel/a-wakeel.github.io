A selection of projects I've worked on professionally and in my spare time.

### DIRBS Core

A large-scale data analytics system to identify counterfeit and non-compliant devices on telecom networks. Originally driven by Qualcomm, I led the continued development including RESTful APIs, ETL pipelines, OLAP batch jobs, and Docker-based CI/CD. Deployed at Pakistan Telecommunication Authority (PTA).

**Tech:** Python, Flask, PostgreSQL, Docker, Redis, StatsD, Grafana

---

### HelloRetail - Pages & Analytics

Sub-products for HelloRetail (Danish e-commerce personalization platform). Developed and maintained the "Pages" and "Analytics" modules as a Java backend developer.

**Tech:** Java

---

### Amazon Review Analyzer

A system to rank Amazon products by detecting real vs. fake reviews. Included a 3NF PostgreSQL database, Flask REST APIs, and an internal IP proxy rotation system that reduced IP blocking by 70%.

**Tech:** Python, Flask, PostgreSQL

---

### VM Memory Analysis Platform

A real-time virtual machine memory analysis system built with Volatility and KVM hypervisor. Integrated with VirusTotal via Apache Kafka for automated threat detection.

**Tech:** Python, KVM, Volatility, Apache Kafka
