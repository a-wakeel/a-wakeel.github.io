## Setting up locally
To setup and run the project locally, follow the steps:

- Clone the repo to your local environment.
- Install a full [Ruby Dev environment](https://jekyllrb.com/docs/installation/)
- Install Jekyll and bundler gems
    ```bash
    gem install jekyll bundler  
    ```
- Change into the project directory and install missing gems
    ```bash
    bundle install
    ```
- Build the site and make it available on local server
    ```bash
  bundle exec jekyll serve  
  ```
